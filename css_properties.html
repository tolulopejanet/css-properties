In CSS, I have learnt that border sets the width, style and color for all four sides of an element's border, box-sizing alters the default CSS box model, color specifies the color of the text of an element, display specifies how an element is displayed onscreen, font-size defines the font size for the text, font-weight specifies the font weight of the text, text-align sets the horizontal alignment of inline content and many more.

The difference between padding, border and margin property is stated below;

Padding - The inner space between the content and the border of your box.
Border - The perimeter of the box. Borders can be invisible or they could be a thick colored line like the green one pictured above.
Margin - The outer space (or lack of space) surrounding the box
